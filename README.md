# Daughter's AI Vocab Buddy ✨

우리 아이를 위한 똑똑한 AI 영어 단어 친구입니다. 사진으로 단어를 스캔하고, 음성으로 퀴즈를 풀며, 구글 드라이브에 안전하게 동기화하세요.

## 🚀 업그레이드된 주요 기능

1. **AI OCR 매직 카메라**: Tesseract.js를 탑재하여 종이 단어장을 사진 찍으면 자동으로 단어와 뜻을 추출합니다.
2. **커스텀 발음 녹음**: 엄마나 아빠의 목소리로 직접 발음을 녹음해 아이에게 들려줄 수 있습니다.
3. **스마트 대시보드**: 일일 스트릭(Streak), 학습 포인트, 마스터 단어 통계를 통해 성취감을 줍니다.
4. **구글 드라이브 동기화**: 학습 데이터를 클라우드에 JSON 형태로 자동 저장하여 기기를 바꿔도 이어서 공부할 수 있습니다.
5. **고급 SRS 알고리즘**: 에빙하우스 망각 곡선을 더 정밀하게 적용하여 최적의 복습 시기를 알려줍니다.

## 🛠 필수 설정 (구글 드라이브 동기화용)

구글 드라이브 동기화 기능을 사용하려면 아래 환경 변수가 필요합니다:

1. [Google Cloud Console](https://console.cloud.google.com/)에서 프로젝트를 생성합니다.
2. Drive API를 사용 설정합니다.
3. OAuth 2.0 클라이언트 ID와 API 키를 발급받습니다.
4. 프로젝트 루트에 `.env.local` 파일을 만들고 아래 내용을 채웁니다:

```env
NEXT_PUBLIC_GOOGLE_CLIENT_ID=여러분의_클라이언트_ID
NEXT_PUBLIC_GOOGLE_API_KEY=여러분의_API_KEY
```

## 📦 새 라이브러리 목록
- `zustand`: 상태 관리
- `tesseract.js`: OCR 이미지 인식
- `framer-motion`: 애니메이션 효과
- `lucide-react`: 아름다운 아이콘
