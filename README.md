# Daughter's AI Vocab Buddy ✨

우리 아이를 위한 똑똑한 AI 영어 단어 친구입니다. 사진으로 단어를 스캔하고, 부모님의 목소리로 퀴즈를 풀며, 구글 시트에 학습 데이터를 기록하세요.

## 🚀 주요 기능

1. **AI OCR 매직 카메라 (정밀 보정)**: 이미지를 흑백/고대비로 자동 보정하여 종이 단어장의 글자를 훨씬 정확하게 읽어옵니다.
2. **커스텀 목소리 퀴즈**: 부모님의 목소리로 직접 발음을 녹음하면, 퀴즈 시간에 로봇 대신 부모님의 목소리가 나옵니다.
3. **스마트 대시보드**: 일일 스트릭(Streak), 학습 포인트, 마스터 단어 통계를 통해 성취감을 줍니다.
4. **구글 시트 실시간 동기화 (GAS)**: Google Apps Script를 연동하여 학습 데이터를 내 구글 스프레드시트로 즉시 전송할 수 있습니다.
5. **자유로운 단어 편집**: 오타나 수정이 필요할 때 목록에서 즉시 단어와 뜻을 고칠 수 있습니다.
6. **고급 SRS 알고리즘**: 에빙하우스 망각 곡선을 적용하여 아이에게 꼭 필요한 복습 시기를 알려줍니다.

## 🛠 필수 설정 (구글 시트 연동용)

구글 시트 동기화 기능을 사용하려면 아래 설정이 필요합니다:

1. [Google Sheets]에서 새 시트를 만듭니다.
2. [확장 프로그램] > [Apps Script]를 클릭합니다.
3. 데이터를 수신할 간단한 `doPost(e)` 스크립트를 작성하고 '웹 앱'으로 배포합니다.
4. 배포된 **웹 앱 URL**을 복사합니다.
5. 앱의 **대시보드 > 구글 시트 동기화** 버튼을 눌러 URL을 입력하고 저장하세요.

## 📦 기술 스택
- `Next.js` (App Router)
- `Zustand`: 상태 관리 및 LocalStorage 유지
- `Tesseract.js`: AI OCR 이미지 인식
- `Framer Motion`: 부드러운 애니메이션
- `Lucide React`: 아이콘 시스템
